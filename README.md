# Matrax

PWA для отслеживания ротации матраса. Подсказывает когда и как переворачивать, ведёт журнал, работает офлайн.

## Что делает

Показывает текущую ориентацию матраса в 3D. Даёт рекомендацию на следующую смену белья: куда повернуть, какой стороной положить. Пишет в журнал дату, время и тип смены.

Поддерживает два типа матрасов:

**Квадратный** (200×200). Цикл из 4 шагов. Каждая смена = flip + поворот на 90° вправо. За полный цикл матрас проходит все 4 стороны света и обе стороны (A/B).

**Прямоугольный** (например, 160×200). Цикл из 2 шагов. Каждая смена = flip + разворот Head↔Foot. Повороты на 90° исключены, матрас не влезет в кровать боком.

## Экраны

Три вкладки внизу.

**Матрас.** 3D модель с компасом, выбор типа матраса, текущее состояние (сторона и направление изголовья). Матрас можно покрутить пальцем, он вернётся на место.

**Смена.** Рекомендация с описанием. Кнопка подтверждения. Экспорт в JSON, сброс.

**Журнал.** Вся история с датой, временем, переходом и иконками действий (flip, rotate, turn).

## Фичи

Тёмная и светлая темы, переключатель в шапке. Настройка запоминается.

Свайп по 3D модели, возврат к актуальному положению при отпускании.

Данные хранятся локально в `localStorage`. Ничего не уходит на сервер.

Работает офлайн через Service Worker.

Тактильный отклик на подтверждении смены (где поддерживается).

## Установка на телефон

1. Открыть https://auskraft.github.io/matrax/ в Chrome
2. Меню (три точки), «Установить приложение»
3. Иконка появится на рабочем столе

Запускается в полноэкранном режиме, неотличимо от нативного приложения.

## Стек

Один HTML файл. Ванильный JS, без сборки, без зависимостей. CSS 3D transforms для матраса. Шрифты Syne и JetBrains Mono из Google Fonts. Иконки PNG 192 и 512.

## Структура

```
matrax/
├── index.html            Приложение целиком
├── sw.js                 Service Worker для офлайна
├── manifest.webmanifest  PWA манифест
├── icon-192.png          Иконка
└── icon-512.png          Иконка для сплэш-экрана
```

## Модель данных

Состояние:
```js
{ shape: 'square' | 'rect', step: number, lastChange: timestamp }
```

Запись журнала:
```js
{
  ts: timestamp,
  shape: 'square' | 'rect',
  from: { side: 'A'|'B', dir: 'N'|'E'|'S'|'W' },
  to:   { side: 'A'|'B', dir: 'N'|'E'|'S'|'W' },
  flipped: boolean,
  rotated: boolean,   // только для square
  turned: boolean,    // только для rect (Head↔Foot)
}
```

Ключи в localStorage: `matrax_state_v3`, `matrax_log_v3`, `matrax_theme_v1`.

## Логика циклов

**Квадратный:**

| Шаг | Сторона | Изголовье |
|-----|---------|-----------|
| 1   | A       | Север     |
| 2   | B       | Запад     |
| 3   | A       | Юг        |
| 4   | B       | Восток    |

После шага 4 возврат к шагу 1.

**Прямоугольный:**

| Шаг | Сторона | Изголовье |
|-----|---------|-----------|
| 1   | A       | Север     |
| 2   | B       | Юг        |

После шага 2 возврат к шагу 1.

## Разработка

Никакой сборки не требуется. Открыть `index.html` в браузере, готово.

Для тестирования PWA возможностей (Service Worker, установка) нужен HTTPS или localhost. Можно поднять простой сервер:

```bash
python3 -m http.server 8000
```

Потом открыть http://localhost:8000.
